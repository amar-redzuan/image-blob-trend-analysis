# Image Blob Trend Analysis
This project combines computer vision and trend analysis to see the performance for the usage of online learning Coddy platform over time.

This project:
- Detects orange blobs using OpenCV
- Counts blobs per image
- Extracts month-year from filename
- Builds time-series trend using pandas
- Visualizes trend with matplotlib
